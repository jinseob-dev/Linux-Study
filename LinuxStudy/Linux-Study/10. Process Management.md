**Understanding Processes**
A process is a running program on your Linux system. Every time you execute a command, open an application, or run a script, Linux creates a process for it.  
Each process gets a unique identifier called a Process ID or <mark>PID</mark>. Think of it like a name tag that helps the system keep track of what's running.  
Processes have a parent-child relationship. When you open a terminal, it creates a shell process. Any command you run from that terminal becomes a child process of the shell.
