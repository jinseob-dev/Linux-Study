# **Introduction to nano**

Text editors are essential tools for working with files in Linux. The <mark>nano</mark> editor is perfect for beginners because it displays helpful shortcuts right on the screen.  
To open <mark>nano</mark>, simply type:  
`nano`  
To edit existing file:  
`nano filename.txt`  
When <mark>nano</mark> opens, you'll see the main editing area and shortcuts at the bottom. The <mark>^</mark> symbol means hold the Ctrl Key. For example, <mark>^X</mark> means Ctrl+X.  

**Basic Navigation**

Use arrow keys to move around the text. The cursor shows your current position.  

**Essential Shortcuts**

The most important shortcuts are displayed at the bottom:  
- <mark>^X</mark> (Ctrl+X): Exit nano
- <mark>^O</mark> (Ctrl+O): Save the file (WriteOut)
- <mark>^W</mark> (Ctrl+W): Search for text
- <mark>^K</mark> (Ctrl+K): Cut current line
- <mark>^U</mark> (Ctrl+U): Paste cut text

**Creating Your First File**

Let's create a simple text file:  
`nano file.txt`  
Type some text, then press <mark>Ctrl+O</mark> to save. You'll see a prompt asking for the filename. Press Enter to confirm, then <mark>Ctrl+X</mark> to exit.  
The beauty of <mark>nano</mark> is its simplicity. Everything you need is. visible on screen, making it ideal for quick edits and learning Linux file editing basics.  

**Using vim Basics**

The <mark>vim</mark> editor is powerful but different from typical text editors. It has two main modes: Normal mode for navigation and Insert mode for typing text.  

**Starting vim**

Open <mark>vim</mark> with:  
`vim`  
Or edit a specific file:  
`vim filename.txt`  

**Understanding Modes**

When <mark>vim</mark> starts, you're in Normal mode. You cannot type text directly. To enter Insert mode, press <mark>i</mark>. You'll see <mark>-- INSERT --</mark> at the bottom.  

**Insert Mode*

In Insert mode, type text normally.  
1. Press <mark>i</mark> to enter Insert mode
2. Type your text
3. Press <mark>Esc</mark> to return to Normal mode

**Basic Navigation in Normal Mode**
- <mark>h</mark>: Move left
- <mark>j</mark>: Move down
- <mark>k</mark>: Move up
- <mark>l</mark>: Move right
Arrow keys also work, but vim users prefer these letter keys.  

**Essential Commands**

All commands work in Normal mode:  
- <mark>i</mark>: Enter Insert mode
- <mark>Esc</mark>: Return to Normal mode
- <mark>:w</mark>: Save file
- <mark>:q</mark>: Quit vim
- <mark>:wq</mark>: Save and quit
- <mark>:q!</mark>: Quit without saving  

**Your First vim Session**

`vim practive.txt`
Press <mark>i</mark>. type some text, press <mark>Ese</mark>, then type <mark>:wq</mark> and press Enter.  
While <mark>vim</mark> seems complex initially, these basics will get you started with file editing.  


# **Editing Files with gedit**

The <mark>gedit</mark> text editor provides a familiar graphical interface similar to Windows Notepad but with more features. It's perfect when you're using a desktop environment.  

**Starting gedit**

Launch <mark>gedit</mark> from the terminal:  
`gedit`  
Open a specific file:  
`gedit filename.txt`  
You can also open multiple files:
`gedit file1.txt file2.txt`  

**Interface Overview**

The <mark>gedit</mark> window contains:
- Menu bar with File, Edit, View oprions
- Toolbar with common buttons
- Main editing area
- Status bar showing line numbers and cursor position

**Basic Editing Features**

Unlike terminal editors, <mark>gedit</mark> works like any modern text editor:  
- Click anywhere to position the cursor
- Type normally to insert text
- Use Ctrl+C, Ctrl+V for copy and paste
- Drag to select text
