# **Introduction to nano**

Text editors are essential tools for working with files in Linux. The <mark>nano</mark> editor is perfect for beginners because it displays helpful shortcuts right on the screen.  
To open <mark>nano</mark>, simply type:  
`nano`  
To edit existing file:  
`nano filename.txt`  
When <mark>nano</mark> opens, you'll see the main editing area and shortcuts at the bottom. The <mark>^</mark> symbol means hold the Ctrl Key. For example, <mark>^X</mark> means Ctrl+X.  

**Basic Navigation**

Use arrow keys to move around the text. The cursor shows your current position.  

**Essential Shortcuts**

The most important shortcuts are displayed at the bottom:  
- <mark>^X</mark> (Ctrl+X): Exit nano
- <mark>^O</mark> (Ctrl+O): Save the file (WriteOut)
- <mark>^W</mark> (Ctrl+W): Search for text
- <mark>^K</mark> (Ctrl+K): Cut current line
- <mark>^U</mark> (Ctrl+U): Paste cut text

**Creating Your First File**

Let's create a simple text file:  
`nano file.txt`  
Type some text, then press <mark>Ctrl+O</mark> to save. You'll see a prompt asking for the filename. Press Enter to confirm, then <mark>Ctrl+X</mark> to exit.  
The beauty of <mark>nano</mark> is its simplicity. Everything you need is. visible on screen, making it ideal for quick edits and learning Linux file editing basics.
