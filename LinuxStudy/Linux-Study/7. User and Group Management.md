
# **Understanding Users and Groups**

Linux is a multi-user operating system, meaning multiple people can use the same computer simultaneously. To organise and control access, Linux uses users and groups.  

**What is a User?**

Every person who uses a Linux system has a user account. Each user has:  
- A unique username (like "alice" or "sarah")
- A user ID number (UID)
- A home directory (usually <mark>/home/username</mark>)
- A default shell program
The system creates some special users automatically. The most important thing is <mark>root</mark>, which has complete control over the system.  

**What is a Group?**

Groups are collections of users. They make it easier to manage permissions for multiple people at once. Instead of giving permissions to each user individually, you can assign permissions to a group.
